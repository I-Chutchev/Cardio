Описание проекта:
У нас есть файл 'train', в котором содержиться проведенное исследование в области кардиологического исследования.
Проведем анализ данных и построим модели машинного обучения. Выберем лучшую и спрогнозируем вероятность сердечного приступа на файле 'test'.

Используемые модели:
DecisionTreeClassifier
RandomForestClassifier
